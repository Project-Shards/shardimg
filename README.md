## 🖥️ Installation

Install shardimg with meson & ninja

```bash
  meson setup _build
  ninja -C _build install
```

## 🙋 Usage/Examples

### Initialize a new image
```bash
shardimg init myCustomImage
```

### Build a new image and use a custom repository
```bash
shardimg build --repo /var/repo
```
